# 💬 千牛聊天记录导出器

一款专为千牛网页版设计的油猴脚本，可自动导出所有客户聊天记录。

## 🚀 核心功能

- 自定义设置：可设置导出客户数量、切换聊天等待时间

- 批量导出：自动遍历左侧客户列表，抓取对应聊天记录

- Excel 导出：自动生成 `.xlsx` 文件

- 分客户工作表：导出后每一个客户一个工作表

- 导出汇总：自动生成导出汇总表，方便查看成功和失败情况

- 实时进度：显示当前导出进度、完成数量、导出日志

- 支持停止：导出过程中可停止任务

## 🛠️ 环境要求

- 🌐 浏览器：Edge 最新版

- 🧩 插件：Tampermonkey 油猴插件

- 💻 网页：千牛网页版

## 📥 安装步骤

1. 🧩 安装 Tampermonkey 插件

   - Edge：扩展商店搜索 `Tampermonkey`

2. 🔓 打开 Tampermonkey 插件

3. ➕ 点击 `添加新脚本`

4. 🧹 删除默认代码

5. 📋 粘贴本项目提供的脚本代码

6. 💾 点击保存

## 📖 使用方法

1. 🌐 打开千牛网页版聊天页面-接待工具-聊天记录查询

2. ⏳ 选择日期-点击查询-等待客户列表加载完成

3. ⚙️ 设置导出数量和等待时间(推荐使用默认1500ms)

4. 🖱️ 点击页面左侧侧的 `开始导出 Excel` 按钮

5. 📥 等待导出完成后，自动下载聊天记录文件

## 📄 导出文件说明

| 工作表 | 说明 |

|---|---|

| 导出汇总 | 记录每个客户的导出状态、消息数、失败原因、导出时间 |

| 客户工作表 | 每一个客户单独生成一个工作表 |

## 📊 客户工作表字段

| 字段 | 说明 |

|---|---|

| 客户名 | 当前导出的客户名称 |

| 消息序号 | 当前客户聊天记录的消息顺序 |

| 发送时间 | 消息发送时间 |

| 发送内容 | 聊天内容、链接、系统提示等 |

| 发送账号 | 发送该消息的账号 |

| 账号类型 | 客服或客户 |

## ⚠️ 注意事项

- 请在聊天记录查询页面使用

- 如果没有出现导出面板，请确认是否进入了聊天记录查询内层页面

- 导出过程中不要刷新页面

- 如果聊天记录加载较慢，可以增加切换等待时间

- 如果历史记录不完整，可以增加向上滚动次数
